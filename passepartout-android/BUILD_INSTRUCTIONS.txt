PASSEPARTOUT ANDROID APP - BUILD INSTRUCTIONS
=============================================

WHAT YOU NEED:
1. Java JDK 17 (free) - https://adoptium.net/temurin/releases/?version=17
   Download: Windows x64 .msi installer
   
2. Android command line tools (free, ~100MB)
   https://developer.android.com/studio#command-tools
   Download "Command line tools only" for Windows

SETUP (one time only):
----------------------

1. Install Java JDK 17 (run the .msi, accept defaults)

2. Create folder: C:\android-sdk

3. Unzip the Android command line tools into: C:\android-sdk\cmdline-tools\latest\
   (so you have C:\android-sdk\cmdline-tools\latest\bin\sdkmanager.bat)

4. Open Command Prompt and run:
   set ANDROID_HOME=C:\android-sdk
   C:\android-sdk\cmdline-tools\latest\bin\sdkmanager.bat "platform-tools" "platforms;android-34" "build-tools;34.0.0"
   (Accept the license by typing 'y' when prompted)

BUILD THE APK:
--------------

1. Unzip this project to a folder, e.g. C:\passepartout-android\

2. Open Command Prompt in that folder:
   cd C:\passepartout-android

3. Set environment (do this each time you open a new Command Prompt):
   set ANDROID_HOME=C:\android-sdk
   set PATH=%PATH%;%ANDROID_HOME%\platform-tools

4. Build the APK:
   gradlew.bat assembleDebug

5. The APK will be at:
   app\build\outputs\apk\debug\app-debug.apk

INSTALL ON YOUR PHONE:
----------------------

Option A (USB):
1. On your phone: Settings -> Developer Options -> USB Debugging (turn on)
   (To enable Developer Options: Settings -> About Phone -> tap "Build Number" 7 times)
2. Connect phone via USB
3. Run: adb install app\build\outputs\apk\debug\app-debug.apk

Option B (File transfer):
1. Copy app-debug.apk to your phone
2. Open it on your phone (you may need to allow "Install unknown apps" in Settings)

FIRST TIME USE:
---------------

1. Open Passepartout app on your phone
2. You need your PHPSESSID cookie. To get it:
   - Log into https://creationeer.co.uk/goforit/ in Chrome on your phone
   - Go to https://creationeer.co.uk/goforit/push-debug.php
   - Open Chrome menu -> More tools -> Developer tools (if available)
   - OR: on a desktop computer, log in, open DevTools -> Application -> Cookies
     and copy the PHPSESSID value
3. Paste the PHPSESSID into the app and tap "Save & Schedule 06:00 Alarm"
4. Tap "Test: Show Task Screen Now" to verify it works
5. The alarm will fire every morning at 06:00 and show a full-screen task

NOTE: The app needs to stay installed and the alarm permission granted.
If you restart your phone the alarm reschedules automatically (BOOT_COMPLETED).
